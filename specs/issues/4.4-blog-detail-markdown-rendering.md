## Description
Create the blog post detail page which fetches data from Supabase based on the URL slug and renders the content using `markdown-it`. The page should also display metadata like tags and publish date, and handle cases where the slug is invalid.

---

## Acceptance Criteria
- [x] Navigating to `/blog/:slug` fetches the correct post from Supabase `posts` table.
- [x] Markdown content is rendered safely and accurately using `markdown-it`.
- [x] Styling for markdown content is handled (e.g., using `@tailwindcss/typography`).
- [x] Metadata (title, excerpt, publish date, tags) is displayed correctly.
- [x] Shows a 404 state or redirects if the post slug is not found.

## Step-by-step Guide

**Files to create/modify:**
- `src/pages/BlogDetail.vue` — Main page for blog details.

**Key decisions:**
- Use `markdown-it` for rendering (already installed).
- Use `prose` (Tailwind Typography) for styling the markdown container.
- Fetch data in `onMounted` or using a watcher if slug changes.

**Non-obvious snippets**:
```vue
<script setup>
import MarkdownIt from 'markdown-it';
const md = new MarkdownIt();
const renderedContent = computed(() => md.render(post.value?.content || ''));
</script>

<template>
  <div class="prose dark:prose-invert" v-html="renderedContent"></div>
</template>
```
