## Portfolio
Build a website for portfolio and blog

## Techstacks

| Side | Tech |
| --- | --- |
| Frontend | Vue 3 (Composition API) + Vite |
| Styling | Tailwind CSS |
| Backend / Database | Supabase (PostgreSQL, Auth, Storage) |
| Deploy | Vercel |


## Screens

### 1. Home

* Hero section: name, tagline, avatar, CTA
* Featured projects (3–4 highlighted projects)
* Latest blog posts (2–3 posts)
* Tech stack / skills (icon grid or marquee)

### 2. About

* Detailed bio
* Timeline of work experience / education
* Categorized skills (Frontend, Backend, Tools…)
* CV link (PDF download from Supabase Storage)

### 3. Projects

* Grid/list of projects with thumbnail, name, short description, tech tags
* Filter by tag/technology
* Detail page: description, demo images/videos, GitHub link, live demo link

### 4. Blog

* Post list: title, excerpt, published date, tags
* Pagination or infinite scroll
* Detail page: markdown content rendered to HTML, social sharing
* Search & filter by tags

### 5. Contact

* Contact form (name, email, message) — stored in Supabase
* Social media links (GitHub, LinkedIn, Twitter…)
