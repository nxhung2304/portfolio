## Step-by-step Guide

**Files to create/modify:**
- `src/pages/ProjectDetail.vue` — Add `useHead` for dynamic SEO.
- `src/pages/Projects.vue` — Add featured filtering logic.
- `src/components/ProjectCard.vue` — Add featured status indicator.
- `supabase/seed.sql` (or similar) — Populate sample data.

**Key decisions:**
- Use `useHead` from `@vueuse/head` to manage `<head>` tags dynamically.
- Featured filter should be integrated into the existing filter UI from Task 3.2.

**Non-obvious snippets**:
```ts
// src/pages/ProjectDetail.vue
import { useHead } from '@vueuse/head'

// Inside script setup after fetching project
useHead({
  title: computed(() => `${project.value?.title} | Portfolio`),
  meta: [
    { name: 'description', content: computed(() => project.value?.description) },
    { property: 'og:image', content: computed(() => project.value?.thumbnail_url) }
  ]
})
```
